Project Overview

The Problem

Solar-powered irrigation pumps face constant volatility. [cite_start]Supply-side fluctuations (clouds) and load-side disturbances (pipe clogs) cause "flow sag," reducing agricultural output and efficiency[cite: 7, 8].

The Solution

We developed a Predictive Adaptive PI Controller. [cite_start]Unlike standard reactive controllers, our solution uses physics-based inverse dynamics to "predict" the voltage needed to reject disturbances instantly[cite: 81].

Control Architecture

The system utilizes a Two-Degree-of-Freedom (2-DOF) architecture. [cite_start]The feedforward path fights the enemy (load disturbance) while the feedback path ensures zero steady-state error[cite: 80, 227].

Key Equations

Feedforward Law (Inverse Dynamics):
[cite_start]V_ff = (R/Kt) * (B*w_ref + estimated_load) + Ke*w_ref [cite: 88]

Total Control Input:
[cite_start]u(t) = u_PI(t) + 0.7 * V_ff(t) [cite: 87]

% Predictive PI Control Loop Snippet
e_pred = Q_ref - C*x_pred;
w_req = Q_ref / K_flow;

% Inverse Dynamics (Feedforward)
V_ff = R*((B*w_req + tau_d)/Kt) + Ke*w_req;

% Control Law (Hybrid)
u_pred = max(0, min(24, (Kp*e_pred + Ki*int_pred) + K_ff*V_ff));
